
De Raspberry Pi model 4B wordt het brein van dit systeem.

# Gebruik
Wordt gebruikt als backend, database en web server


# Services
### Backend
De backend communiceert met de slimme opslag, website en de database. Als de website opvraagt hoeveel stukken er van een bepaald onderdeel zijn, dan kijkt de backend in de database en geeft aan de hand daarvan antwoord. Ook wordt het gebruikt om de aantallen aan te passen in de database. Daarnaast is het ook het systeem dat automatisch een nieuwe plek in de database genereert wanneer er een nieuw artikel wordt toegevoegd aan de slimme opslag. Ook houd de backend in de gaten wanneer er te weinig van een onderdeel is en geeft dan aan aan de website dat er een waarschuwing moet komen.
### Database
De database slaat alle gegevens op. Docent Accounts van de website, onderdelen en hun aantallen, waar onderdelen zich bevinden en nog meer.
### Web server
Dit is de server waar de website op gehost wordt.
